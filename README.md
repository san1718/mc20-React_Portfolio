## Overview

Hi, I'm Sung Min, a web developer passionate about building interactive and user-friendly applications. This portfolio showcases my work and skills using React. Here, you'll find projects I've built, my experience, and a way to connect with me.

## Features

- **Dynamic Navigation:** The portfolio features a smooth, React-powered navigation bar that allows users to switch between sections without reloading the page.
- **About Me Section:** Displays a professional photo and a brief bio about the developer.
- **Project Showcase:** Highlights six featured projects with images, titles, and links to live demos and GitHub repositories.
- **Contact Form:** Includes fields for name, email, and message with built-in validation to ensure proper input.
- **Resume Section:** Offers a downloadable resume and lists the developer’s key technical skills.
- **Footer with Social Links:** Displays icons linking to GitHub, LinkedIn, and another platform like Stack Overflow or Twitter.

## Technologies Used

- **React.js** – For building a fast and responsive single-page application.

- **JavaScript (ES6+)** – For handling component logic and interactions.
- **HTML5 & CSS3** – For structuring and styling the portfolio.
